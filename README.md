# Projeto html básico

<p>No início dos anos 2000 eu comecei a estudar html por conta própria. Na época cheguei a construir uma página utilizando um bloco de notas, digitando tag html por tag. Foi muito prazeroso, mas também muito trabalhoso.</p>
<p>Essa página era uma ezine semanal, e eu atualizava todo sábado de manhã. Enviava um e-mail para os usuários cadastrados na página contendo um resumo dos artigos publicados naquela semana. Esses arquivos eram enviados a mim por diversos autores, com os quais fiz contato por e-mail. Sempre falando sobre negócios e comportamento.</p>  
<p>Quando a página atingiu 3 mil visitas menais, e eu comecei a gastar meio dia de trabalho para manter a página atualizada, decidi parar com ela porque estava tomando tempo demais e eu precisava fazer um curso de inglês.</p>
<p>Foi uma pena, mas foi necessário!</p>
<p>A ideia deste projeto é apenas relembrar a estrutura e formatação de uma página web, usando tags html básicas para matar saudades!</p>
